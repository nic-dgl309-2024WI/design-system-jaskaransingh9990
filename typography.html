Utility Class Names

  • u-ff-heading: Font family for headings
  • u-ff-body: Font family for body text
  • u-fs-heading: Font size for headings
  • u-fs-body: Font size for body text
  • u-fw-bold: Font weight for bold text
  • u-fw-normal: Font weight for normal text
  • u-line-height: Line height for text
  • u-letter-spacing: Letter spacing for text

Font Uses

  • h1: Display font for headings
  • h2: Subheading font for headings
  • p: Body font for paragraphs
  • strong: Bold font for emphasized text
